## Target audience

Students, post-doctoral fellows, and researchers interested in acquisition, reconstruction, and analysis of musculoskeletal MRI. Skill level: basic to intermediate.

## Learning objectives

1. Describe the contrasts, speed, artifacts, and utility of sequences used in clinical MSK imaging
2. Describe quantitative imaging markers and techniques used in joints, muscle, and skeleton
3. List best practices in acquisition, segmentation, registration, and fitting for quantitative MSK MRI
4. Discuss common image analysis methods using segmentation and shape analysis tools
5. Discuss emerging applications of low-field MRI and opportunities to build and disseminate these applications
